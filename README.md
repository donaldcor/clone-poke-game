# React + Vite
## How to run this project (local)

These are the minimal steps to run this project on your machine (macOS / zsh). This project uses Vite + React.

1. Install dependencies

```bash
npm install
```

2. Run in development mode

```bash
npm run dev
```

Open the URL printed by Vite (usually http://localhost:5173) in your browser.

Also, the app is deployed on vercel. You can check it out here: https://clone-poke-game.vercel.app
